# Point of Sales (POS) System

A modern, fast, and responsive Point of Sales application built with React, TypeScript, and Vite.

## Features
- **Cashier Dashboard**: Easy-to-use interface for processing transactions.
- **Product Management**: View and select products for sale.
- **Cart System**: Dynamic cart to manage items before checkout.
- **Responsive Design**: Works perfectly on desktops and tablets.
- **Mock API Ready**: Integrated with a local mock API for testing and development.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Build Tool**: Vite
- **HTTP Client**: Axios

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
